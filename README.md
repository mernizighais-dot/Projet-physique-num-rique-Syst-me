# Projet-physique-num-rique-Syst-me

Ce dépôt contient tous les fichiers nécessaires au fonctionnement du projet d'analyse de spectre.

## Contenu du dépôt

*   **main.sh** : Le script principal (Bash) qui lance l'analyse.
*   **recherche_plot.py** : Script Python principal.
*   **intensite.py** : Module Python contenant les fonctions utilisées par `recherche_plot.py`.
*   **Spectre_photoluminescence.txt** : Données brutes utilisées par défaut.

## Utilisation

⚠️ **Important :** Pour que les scripts trouvent les fichiers de données et les modules, vous devez exécuter les commandes **depuis l'intérieur du dossier**.
